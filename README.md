An intermediate Snake game built with Python and Pygame. Control the snake, eat food to grow longer, and avoid crashing into the walls or yourself. The game features smooth grid-based movement, random food spawning, and score tracking. It's an awesome project that demonstrates core game loop logic and collision detection in Python.


Directions to run Snake game: 

git clone https//github.com/Darraneica/snake-game
cd snake-game
pip install -r requirements.txt
python main.py
